# Guía Rápida de Hacking Ético

Esta es una nota de prueba simulando un archivo de **Obsidian**.
Aquí exploramos conceptos básicos de ciberseguridad.

## 1. Fase de Reconocimiento
El reconocimiento o *footprinting* es el primer paso. Algunas herramientas comunes:
- Nmap
- Shodan
- Recon-ng

### Ejemplo de escaneo básico
Aquí tienes un ejemplo de cómo usar Nmap:

```bash
nmap -sC -sV 192.168.1.100
```

## 2. Análisis de Vulnerabilidades
Una vez obtenidos los puertos abiertos, buscamos CVEs asociados.
> Siempre verifica los exploits antes de lanzarlos.
